# 联系方式

Kube-OVN 社区欢迎所有用户和开发者参与，可通过下面的方式与我们沟通：

## 在线交流

- Slack：在 [CNCF Slack](https://slack.cncf.io) 加入后访问 [#kube-ovn](https://cloud-native.slack.com/archives/C0AQMHUDX39) 频道
- LinkedIn：[Kube-OVN 官方主页](https://www.linkedin.com/company/kube-ovn/)
- Issue / 讨论：在 [GitHub Issues](https://github.com/kubeovn/kube-ovn/issues) 反馈问题，[GitHub Discussions](https://github.com/kubeovn/kube-ovn/discussions) 参与设计讨论
